# 程序简介
受七龙珠启发，项目命名为万能胶囊，旨在于实现附件服务器所需所有功能。  
万能胶囊是日本著名漫画家鸟山明所著漫画《龙珠》中布尔玛家族企业所研制开发的一种高科技产品，小小的胶囊可以装下摩托、汽车、飞机和房屋等。  

# 运行环境
.NET 6.0

# 开箱即用
直接运行服务即可快速搭建附件服务器。

# 优势
无需任何中间件，便可快速实现附件服务器。
